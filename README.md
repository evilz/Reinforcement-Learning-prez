# Reinforcement Learning – présentation

Ce dépôt contient une présentation (HTML/CSS/JS) basée sur **reveal.js**.

## Utilisation

- Ouvrir `index.html` directement dans un navigateur, ou
- Servir le répertoire via un serveur HTTP local (recommandé) :

```bash
python3 -m http.server 8000
```

Puis ouvrir : http://localhost:8000/

## Développement

Les fichiers principaux :

- `index.html` : contenu des slides
- `reveal.js` / `reveal.css` : bibliothèque Reveal
- `theme/` : thèmes
